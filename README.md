
# CMP - Customizable Management Platform

## Installation

Install meteor METEOR@2.7.3

Install node dependencies

```bash
  npm install
```

And run web server with 
```bash
  npm run start
```
You can watch and recompile SCSS with 
```bash
  npm run watch
```
